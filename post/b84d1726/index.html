Mac 实用软件推荐

以下是我使用 Mac 时觉得比较好用的一些插件，由于本人是第一次使用 Mac，所以不一定适用于所有人。

1. Maccy「剪切板历史」

可以记录剪切板历史，类似 Windows 的 Win + V 快捷键，但不同的是提供了搜索功能（支持正则表达式），同时也能够保存复制的文件。

UI 比较简约，但很好用。

安装使用

  1. 可以在 App Store 下载（需要付费）
  2. 前往 Github 仓库，参考教程进行安装（提供了 brew 指令安装和安装包下载等方式）

安装成功并启动后，按 Shift + Command + C 可以呼出 Maccy 的界面，选择想要粘贴的内容后，再 Command + V 即可粘贴剪切板历史的指定内容。

建议在界面的 “偏好设置 - 通用” 里勾选 “登录时启动”，就不用每次开机手动启动了。

2. Mos「鼠标触控板控制」

能够单独控制鼠标和触控板的滚轮反转（自然滚动）功能。

一般情况下，我们想实现触控板双指下滑上翻页、上滑下翻页；鼠标滚轮前滚上翻页、后滚下翻页。但由于 Mac 中鼠标和触控板的行为逻辑是绑定在一起的，所以只能达到其中一种效果，不能同时实现。

安装使用

  1. 前往官网下载

安装启动后，需要在「设置 - 隐私与安全性 - 辅助功能」中允许 Mos 控制电脑。

里面默认勾选了 “平滑滚动”，但用蓝牙连罗技鼠标时会出现按住 Shift 滑动滚轮无法调节横向滚动条的问题，可以取消勾选。

3. Background Music「音量控制」

单独控制程序的音量大小，类似 Windows 的音量合成器。

安装使用

安装启动后，在屏幕右上方菜单栏可以看到图标，点击即可调节程序音量。

似乎无法对 zoom 这类会议软件的音量进行控制。

4. Karabiner-Elements「键盘映射」

能够实现 Mac 的键盘映射。例如：将 Fn 键映射为 Cmd 键（单向映射，Cmd 键依然能作为 Cmd 键使用）。

以前使用 Windows 第一次用 Mac 的人群，可以使用 Fn → Cmd 映射，再添加右 Option 键 → Fn 的映射（用来组合 F1-F12 功能键）。

安装使用

  1. 前往官网下载

安装启动后，根据提示打开对应权限，然后在左边菜单栏的「Simple Modifications → For all devices」中点击「Add item」添加键盘映射。

启动程序后会将已修改过的「修饰键配置」恢复至默认，添加的键盘映射程序关闭后依然生效。

5. IINA「音视频播放器」

功能完善的音视频播放器，界面简洁，支持多种格式。

安装使用

  1. 前往官网下载
