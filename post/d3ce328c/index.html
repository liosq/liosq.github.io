应用软件

记录一些实际生活中用到的比较好用的应用软件，方便自己在以后更换设备时迁移应用和数据信息等等

移动端

滴答清单

很轻松地添加待办事项，对于我这种比较健忘的人来说很有帮助

软件有收费功能（日历式显示待办、代办设置时间段、关联微信等等），但免费功能就已经能满足大部分的需求了

flomo

记笔记的应用，不像滴答清单一样能通知提醒，但是用来当一个便签一样记录下自己随笔也是不错的

也是有收费功能（每日回顾、反向链接、密码锁等等），也是一样，对我来说免费的就足够了

为什么不用系统自带便签

因为我不确定会不会一直使用这个品牌的手机，数据同步不好搞，虽然随笔可能不是很重要，但是既然能数据同步为什么不同步呢

小黑盒

可以在手机上看 steam 游戏的实时价格

LocalSend

可以让局域网内的多设备相互传输文件

如果电脑无法接收文件，可以尝试以下步骤：

  1. 进入 控制面板\系统和安全\Windows Defender 防火墙 界面
  2. 点击高级设置，选择入站规则
  3. 找到 LocalSend，并将其启用（即防火墙放行）

桌面端

Spotify

  1. 能免费听国内付费歌曲（至少目前接触到的像周总的歌都能听）
  2. 需要科学上网

国内平台的歌单迁移，可以参考 迁移 网易云 / QQ 音乐到 Apple Music, Youtube Music, Spotify | yyrcd

Lyricify

Github 开源项目 GitHub - WXRIW/Lyricify-App: Lyricify

适配 Spotify 显示桌面歌词，

Snipaste

截屏软件

  1. 按 F1 可以快速截屏
  2. 可以把截屏图片固定在屏幕上

PowerToys

微软推出的小工具合集，包含：

  1. 速览（类似 QuickLook）
  2. OCR 识别
  3. 快速置顶窗口
  4. 自定义重命名文件

等小插件

微软输入法

  1. 支持 U/V 模式
  2. UI 比较简洁（微信输入法 UI 也不错，但是不支持 U 模式）

微软的时间日期是分开的，如果想输入 “sj” 时出现当前时间戳（年月日时分秒），可以参考该博客Windows10 微软输入法快速输入时间的格式 - 雁书几封 - 博客园 (cnblogs.com)
